# 모의 면접 평가 명령

3-에이전트 토론 시스템을 통해 모의 면접 결과를 평가합니다.

## 사용법
```
/evaluate [질문지파일경로]
```

질문지 파일에는 각 질문에 대한 답변이 기록되어 있어야 합니다.

## 실행 절차

### 1단계: 평가 자료 수집
1. 이력서 파일 읽기 (`resumes/[이름].md`)
2. 질문지 및 답변 기록 읽기 (`resumes/[이름]_questionnaire.md`)
3. (선택) 포트폴리오/GitHub 정보

### 2단계: 기술 평가자 분석 🔬
`.claude/skills/evaluator-technical.md` 스킬 참조

```markdown
## 기술 평가자 분석 시작

### 평가 기준
- 엄격하고 비판적 관점
- 증거 기반 평가
- 3년차 기대 수준 엄격 적용

### 분석 항목
1. JavaScript/TypeScript 역량 (/25)
2. 프레임워크 숙련도 (/25)
3. 웹 성능 및 최적화 (/20)
4. 코드 품질 및 아키텍처 (/15)
5. 문제 해결 능력 (/15)

### Critical Issues 확인
- [ ] 기본 개념 설명 불가
- [ ] 프레임워크 핵심 동작 이해 부족
- [ ] 기술 선택 근거 설명 불가
- [ ] 성능 최적화 경험 전무
```

### 3단계: 커뮤니케이션 평가자 분석 💬
`.claude/skills/evaluator-communication.md` 스킬 참조
