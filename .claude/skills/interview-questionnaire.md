# 면접 질문 생성 스킬

## 역할
프론트엔드 개발자 3년차 이직 면접을 위한 맞춤형 질문지를 생성합니다.

## 질문 구성 원칙

### 연계형 질문 구조
하나의 주제에 대해 깊이 파고들 수 있도록 설계합니다.

```
기본 질문 → 심화 질문 → 경험 질문 → 상황 대응 질문
```

예시:
1. "React의 Virtual DOM이 무엇인가요?"
2. "실제 DOM 업데이트 과정을 설명해주세요"
3. "Virtual DOM으로 인한 성능 이슈를 경험한 적 있나요?"
4. "대량의 리스트 렌더링 시 어떻게 최적화하시겠어요?"

---

## 1. 기술 질문 (JavaScript/TypeScript)

### 필수 질문
1. **클로저(Closure)**
   - 클로저가 무엇인지 설명해주세요
   - 클로저를 활용한 실제 코드 예시를 들어주세요
   - 클로저로 인한 메모리 누수 경험이 있나요?

2. **이벤트 루프**
   - JavaScript 이벤트 루프를 설명해주세요
   - microtask와 macrotask의 차이점은?
   - 비동기 처리 시 우선순위 문제를 겪은 적 있나요?

3. **비동기 처리**
   - Promise와 async/await의 차이점
   - Promise.all vs Promise.allSettled vs Promise.race
   - 에러 처리 전략은 어떻게 하시나요?

4. **TypeScript**
   - TypeScript를 사용하는 이유는?
   - Generic을 활용한 경험을 설명해주세요
   - Type과 Interface의 차이와 선택 기준

---

## 2. 프레임워크 질문 (React/Vue 중심)

### React 질문
1. **상태 관리**
   - useState vs useReducer 선택 기준
   - 전역 상태 관리 도구 선택 경험
   - 상태 정규화(normalization) 경험

2. **성능 최적화**
   - useMemo, useCallback, React.memo 사용 기준
   - 리렌더링 최적화 경험
   - Suspense와 lazy loading 활용

3. **Hooks 심화**
   - 커스텀 훅을 만든 경험
   - useEffect 의존성 배열 관리
   - useRef 활용 사례

### Vue 질문
1. **반응성 시스템**
   - Vue 3 Composition API vs Options API
   - ref와 reactive의 차이
   - computed vs watch vs watchEffect

2. **상태 관리**
   - Pinia/Vuex 사용 경험
   - provide/inject 활용

---

## 3. CSS/스타일링 질문

1. **레이아웃**
   - Flexbox vs Grid 선택 기준
   - 반응형 디자인 구현 전략

2. **성능**
   - 리플로우(Reflow)와 리페인트(Repaint)
   - CSS-in-JS vs CSS Modules 선택 이유
   - Critical CSS 최적화 경험

3. **실무**
   - 디자인 시스템 구축 경험
   - 다크모드 구현 방법
   - 애니메이션 최적화 (GPU 가속)

---

## 4. 웹 성능 질문

1. **측정 및 분석**
   - Core Web Vitals (LCP, FID, CLS) 설명
   - 성능 측정 도구 사용 경험
   - 실제 성능 개선 사례

2. **최적화 기법**
   - 번들 사이즈 최적화 방법
   - 이미지 최적화 전략
   - 코드 스플리팅 적용 경험

---

## 5. 협업/커뮤니케이션 질문

1. **코드 리뷰**
   - 코드 리뷰 시 중점적으로 보는 부분
   - 리뷰에서 의견 충돌 시 해결 방법

2. **협업 경험**
   - 백엔드/디자이너와 협업 시 갈등 해결 사례
   - API 스펙 변경 대응 경험

3. **성장**
   - 최근 학습한 기술과 이유
   - 주니어 온보딩/멘토링 경험
